
# Smart Medicine Reminder System

## Project Overview
This project is developed using Python.
It helps users remember medicine timings using:
- Alarm Sound
- Desktop Notifications
- SQLite Database

---

## Features
- Add Medicines
- Store Data in Database
- View Medicine Records
- Desktop Notification
- Alarm Reminder

---

## Technologies Used
- Python
- SQLite
- Plyer
- Playsound

---

## Required Libraries

Install libraries using:

pip install plyer
pip install playsound==1.2.2

---

## How to Run

1. Open terminal
2. Go to project folder
3. Run:

python main.py

---

## Real-Life Problem
Many people forget to take medicines on time.
This project helps users manage medicine schedules properly.
